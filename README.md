# Online Banking Management System
## Overview
Online Banking Management System is a web-based application developed using Java, Servlets, JDBC, and MySQL.  
It allows users to log in and perform banking operations like deposit, withdrawal, and fund transfer.
##  Technologies Used
- Java
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat
##  Features
- User Login Authentication
- Deposit Money
- Withdraw Money
- Fund Transfer
- Database Connectivity using JDBC
- Balance Validation
## Project Architecture
Frontend (HTML/CSS) → Servlets → JDBC → MySQL Database
##  How to Run
1. Create database using `database.sql`
2. Import project into Eclipse (Dynamic Web Project)
3. Add MySQL Connector JAR
4. Deploy on Tomcat
5. Run on `http://localhost:8080/Online-Banking-Management-System`
